# Sales & Revenue Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance, revenue trends, top-performing products, and regional sales distribution.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Features
- KPI Cards (Sales, Profit, Quantity, Orders)
- Revenue Trend Analysis
- Top Products Analysis
- Category Analysis
- Regional Sales Analysis
- Interactive Slicers (Region, Category, Year)

## Dashboard Preview

![Dashboard](dashboard.png)

## Key Insights
- Sales increased steadily from 2015 to 2017.
- Technology generated the highest sales.
- Top products contributed significantly to revenue.
- Regional analysis highlights sales distribution across markets.
